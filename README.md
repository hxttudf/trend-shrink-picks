# 趋势缩量选股 (Trend Shrink Picks)

每日15:22自动运行，基于回测优化策略筛选A股趋势缩量标的。

## 策略逻辑

```
优化策略: 趋势票 + 距MA20(10-20%) + 量比<0.3 + 20日涨5-25%
买入: T+1开盘 | 卖出: T+N收盘
```

**条件：**
1. 多头排列: close > MA20 > MA60
2. 距MA20 10%-20%（强势趋势中）
3. 量比 < 0.3（极致缩量 = 主力控盘/惜售）
4. 20日涨幅 5%-25%

| N20    | +10.84% | 0.269  |
