# 趋势缩量选股 (Trend Shrink Picks)

每日15:22执行，选出A股市场**趋势+缩量**信号。

## 策略逻辑

> **大道至简：趋势票 + 缩量回调 + 中段涨幅**

当一只股票处于上升趋势（close > MA20 > MA60）中，出现缩量回调且涨幅适中时入场。

```
条件:
  close > MA20 > MA60          ← 多头排列
  距MA20: 10-20%               ← 回调到均线附近
  量比: < 0.3                  ← 极度缩量（无人问津时）
  20日涨幅: 5-25%              ← 刚启动，未过热
  不刻意排除ST                 ← ST也有好票
```

## 回测表现

| 指标 | 值 |
|------|-----|
| 回测区间 | 2024-01 ~ 2026-07 |
| 总信号 | 162 |
| 交易日 | 137 |
| N20 avg | +6.52% |
| 上涨率 | ~50% |
| 最大单票 | +128% (20日) |

**特点：** 信号稀疏（日均~1只），但出现时的赔率结构好。少数爆款贡献主要收益。

## 数据

选股结果存放在 `/home/ubuntu/databases/trend_picks.db`，Datasette port 8001 可查。

表结构:
- `daily_picks` — 每日选股明细 + T+N收益
- `daily_summary` — 每日汇总

## 运行

```bash
./run.sh   # cron wrapper
```

定时: `22 15 * * 1-5` (交易日15:22)
