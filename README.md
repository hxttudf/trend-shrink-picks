# 趋势缩量选股 (Trend Shrink Picks)

每日15:22执行，4个策略同时跑。

## 策略

| 策略名 | 距MA20 | 量比 | 20日涨 | T20均值 | 涨率 | 日均信号 |
|--------|--------|------|--------|---------|------|---------|
| **原版** | 10-20% | <0.3 | 5-25% | +6.9% | 51% | 1.3 |
| **极品A** | 12-25% | 0.1-0.3 | 3-15% | +9.1% | 54% | 1.1 |
| **极品B** | 12-25% | <0.3 | 3-15% | **+14.2%** | **59%** | 1.1 |
| **超缩量** | 10-20% | <0.15 | 3-15% | +14.1% | 58% | 1.1 |

核心条件：`close > MA20` 多头排列，缩量不追高，不排ST。

## 数据结构

DB: `/home/ubuntu/databases/trend_picks.db`

- `strategies` — 4个策略配置
- `daily_picks` — 每日选股明细(含strategy_id和策略名关联)
- `daily_summary` — 每日策略汇总

## 查询

```bash
python3 query_picks.py                        # 各策略统计
python3 query_picks.py today                  # 今日选股
python3 query_picks.py list 2026-07-10        # 指定日期
python3 query_picks.py list 2026-07-10 premium_b  # 指定策略
```
