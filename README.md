# 趋势缩量选股 (Trend Shrink Picks)

每日15:22执行，4个策略同时跑，A股趋势+缩量信号。

## 策略

| 策略名 | 距MA20 | 量比 | 20日涨 | T20均值 | 涨率 | 特点 |
|--------|--------|------|--------|---------|------|------|
| **原版** | 10-20% | <0.3 | 5-25% | +6.6% | 47% | 信号最多,兜底 |
| **极品A** | 12-25% | 0.1-0.3 | 3-15% | +14.6% | 59% | 量比精度最高 |
| **极品B** | 12-25% | <0.3 | 3-15% | **+24.4%** | **67%** | 综合最强 |
| **超缩量** | 10-20% | <0.15 | 3-15% | **+26.6%** | **68%** | Sharpe最高 |

核心条件：`close > MA20` 多头排列，缩量不追高，不排ST。

## 数据结构

DB: `/home/ubuntu/databases/trend_picks.db`

- `strategies` — 策略配置表
- `daily_picks` — 每日选股明细 + T+N收益 (含strategy_id)
- `daily_summary` — 每日每个策略的结果汇总

## 查询

```bash
python3 query_picks.py          # 各策略统计数据
python3 query_picks.py today    # 今日选股
python3 query_picks.py list 2026-07-10  # 指定日期
python3 query_picks.py list 2026-07-10 premium_b  # 指定策略
```
